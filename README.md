
# DEVPORT - Professional Developer Portfolio

A modern, high-performance developer portfolio built with Next.js 14, TypeScript, and Tailwind CSS.  
Designed to clearly present technical skills, real-world projects, experience, and credibility.


## Features

- High performance (target: 95+ Lighthouse score)
- Modular and scalable design using Atomic Design methodology
- Type-safe component architecture with TypeScript
- Fully responsive layout for all screen sizes
- SEO-optimized structure and metadata
- Clean UI built with Tailwind CSS
- Fast load time (under 2 seconds goal)
- Modern Next.js 14 App Router structure

  

##  Tech Stack

- **Framework:** Next.js 14 with App Router
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

- **Fonts:** Inter and JetBrains Mono
- **Tooling:** ESLint, Prettier

##  Getting Started

### Prerequisites
- Node.js 18+ 
- npm, yarn, pnpm, or bun

### Installation

1. **Clone the repository**

2. **Install dependencies**
```bash
npm install
```

3. **Run development server**
```bash
npm run dev
```

4. **Open your browser**
Navigate to [http://localhost:3000](http://localhost:3000)

### Available Scripts

```bash
npm run dev          # Start development server
npm run build        # Build for production
npm run start        # Start production server
npm run lint         # Run ESLint
```
